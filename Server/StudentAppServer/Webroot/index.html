Guide till hur man använder min WebService

Olika format:

För att få ut json:

/StudentAPI?format=json

Prova länken: JSON

För att få ut xml:

/StudentAPI?format=xml

Prova länken: XML

Om du vill få ut data på id:

/StudentAPI?format=xml&id=10

Prova länken: XML med id 10

Error response:

Om du sökt på student som inte har några kurser:

{ "courses": [ { "course_code": "No courses added" } ] }
